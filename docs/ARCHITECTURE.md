# Lumina Architecture Rules

## Directory Structure
src/app            → Pages and routes  
src/components     → Reusable UI components  
src/lib            → Utilities  
public/            → Static assets  

## Content Rules (Lumina-specific)
Child-friendly content only.  
Soft palettes.  
No dark themes.  

## Commit Rules
Major changes → feature/branch  
Minor fixes → atomic commits  

## Cursor Agent Guidelines
Only modify files relevant to the task.  
Do not restructure folders unless explicitly told.  
Follow cursor.json rules strictly.
